# Como usar o Cline neste projeto

O Cline deve ler a pasta `.clinerules/`.

## Primeiro prompt recomendado

```txt
Leia todos os arquivos dentro de .clinerules/ antes de começar.

Depois da leitura:

1. Faça um resumo do entendimento do projeto.
2. Identifique possíveis conflitos ou dúvidas.
3. Proponha a próxima etapa.

Não altere arquivos ainda.
```

## Prompt para implementação controlada

```txt
Leia todos os arquivos da pasta .clinerules/.

Implemente somente o módulo solicitado.

Antes de alterar arquivos, apresente o plano.
Após concluir, liste:

- arquivos criados/alterados
- o que foi feito em cada arquivo
- comandos que preciso rodar
- mensagem de commit sugerida

Não execute git push.
```

## Cuidados

Não pedir ao Cline para criar toda a plataforma de uma vez.

Sempre dividir em módulos pequenos.
