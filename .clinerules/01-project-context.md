# Contexto do Projeto — Diagnóstico 360

Este projeto é uma plataforma interna da Kairos para diagnóstico empresarial, CRM, campanhas, planos de ação, relatórios e IA.

O objetivo do MVP é validar o produto internamente antes da comercialização.

## Módulos principais

1. Autenticação
2. Cadastro de clientes/empresas
3. Diagnóstico 360°
4. Dashboard executivo
5. CRM de leads
6. Rastreamento de campanhas
7. Biblioteca de criativos
8. Inteligência Artificial
9. Planos de ação
10. Relatórios

## Fluxo principal do produto

Lead entra → Kairos cadastra cliente → Consultor aplica Diagnóstico 360° → Sistema calcula scores → Dashboard mostra gargalos → IA gera diagnóstico automático → Consultor cria plano de ação → Sistema gera relatório PDF/Excel.

## Prioridade do MVP

Construir primeiro:

1. Login
2. Clientes
3. Diagnóstico 360°
4. Cálculo de score
5. Dashboard executivo
6. Plano de ação
7. Relatórios
8. IA para diagnóstico
9. CRM
10. Integração Meta Ads
11. Integração WhatsApp

Não iniciar integrações externas antes do núcleo do sistema estar funcional.
